# Bitrix24 BI-коннектор — skill для Claude Code

[🇬🇧 English](README.md) | 🇷🇺 Русский

Skill для [Claude Code](https://docs.anthropic.com/en/docs/claude-code), обучающий Claude работе с HTTP API BI-аналитики Bitrix24. С активированным skill'ом Claude умеет получать список таблиц, описывать их колонки и забирать строки с фильтрацией по диапазону дат и по предикатам.

## Как начать

После установки skill'а в начале разговора сообщи Claude URL своего портала Bitrix24 и bi-токен — например:

> Мой портал Bitrix24 — `https://example.bitrix24.ru`, мой bi-токен — `<your-token>`.

Claude будет переиспользовать их во всех запросах текущей сессии, так что указать нужно только один раз. Если bi-токена ещё нет — см. [Получение bi-токена](#получение-bi-токена) ниже.

## Что Claude умеет делать с активным skill'ом

- «Сколько успешных лидов у меня на портале Bitrix24 за последние три месяца?»
- «Покажи сделки со стадией = успешная, созданные за последние 3 месяца.»
- «Перечисли задачи, у которых TITLE содержит 'CRM:' и ID между 21 и 27.»
- «Найди задачи, у которых DEADLINE равен null.»

## Что внутри

- `.claude/skills/bitrix-bi-connector/SKILL.md` — сам skill.
- `BICONNECTOR_PROTOCOL.md` — исходная спецификация API (на русском), источник истины, на котором построен skill.

## Установка skill'а

**Локально в проекте (по умолчанию).** Claude Code автоматически подхватывает `.claude/skills/<name>/SKILL.md` при запуске внутри каталога проекта. Склонируй репозиторий и запусти `claude` в нём — skill активируется на соответствующих запросах.

```bash
git clone <repo-url>
cd <repo-dir>
claude
```

**Глобально (для пользователя).** Скопируй папку skill'а один раз, и он будет доступен во всех твоих проектах.

```bash
cp -r .claude/skills/bitrix-bi-connector ~/.claude/skills/
```

На Windows PowerShell:

```powershell
Copy-Item -Recurse .claude\skills\bitrix-bi-connector $env:USERPROFILE\.claude\skills\
```

## Получение bi-токена

Нужен ключ BI-аналитики из твоего портала Bitrix24. Перейди в *CRM → Аналитика → Оперативная аналитика → BI-аналитика → Настройки BI-аналитики → Управление ключами* (это путь на портале с русской локалью — она в Bitrix24 по умолчанию; в других локалях названия пунктов меню могут отличаться) или открой `https://<твой-портал>.bitrix24.ru/biconnector/key_list.php` напрямую. Токен — это учётные данные: не клади его в систему контроля версий и в логи.

## Лимиты строк на BI-сущность

Битрикс24 ограничивает количество строк, которое BI-коннектор возвращает из одной BI-сущности, в зависимости от тарифа облачного Битрикс24. Актуальные значения см. в [официальном сравнении тарифов](https://www.bitrix24.ru/prices/compare_cloud_plans.php).

| Тариф | Лимит строк |
|---|---:|
| Бесплатный | 0 |
| Базовый | 10 000 |
| Стандартный | 10 000 |
| Профессиональный | 100 000 |
| Энтерпрайз | 250 000 |
